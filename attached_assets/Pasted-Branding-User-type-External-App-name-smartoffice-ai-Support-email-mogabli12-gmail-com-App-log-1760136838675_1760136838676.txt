Scope justifications
Sensitive scopes
SmartOffice AI is an intelligent admin assistant for small businesses and professionals. We use calendar.readonly scope to: - Display upcoming calendar events in the user's dashboard - Include calendar data in automated weekly reports (PDF/Word) - Help users manage their schedules more efficiently The calendar data is only read (view-only access) and displayed back to the user who owns it. We do not store, share, or transmit this data to third parties. Each user connects their own Google Calendar account for their personal use only.
